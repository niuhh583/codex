## AI Image Prompt Structure

Use this order:

1. Subject and action
2. Scene and context
3. Composition and camera
4. Light and color
5. Texture and material
6. Style reference
7. Brand constraints
8. Negative constraints

Do not include copyrighted artist imitation unless the user provides rights or asks for a high-level non-infringing style description.

### Gemini-3.1-Flash-Image-Preview Notes

- Use clear natural-language instructions.
- Specify exact layout, subject relationships, image ratio, and what should not appear.
- If text must appear, keep it short and request clean editable-looking typography; verify text after generation.
- Provide reference constraints as descriptions when no reference files are available.

### Midjourney Notes

- Put the strongest visual nouns and style cues early.
- Include composition, lens/perspective, lighting, material, and mood.
- Use parameters only when useful, such as aspect ratio, stylization, or seed.
- Avoid asking for precise long text inside the image; generate typography separately when accuracy matters.

### GPT-Image-2 Notes

- Give direct art-direction instructions with clear subject, scene, composition, and constraints.
- State brand-safe requirements, forbidden elements, and any text accuracy needs.
- Use iterative refinement language when asking for variants: preserve composition, adjust color, simplify background, improve material realism, etc.

### Cautions

- Check copyright, trademark, font licensing, portrait rights, and competitor similarity.
- Do not imitate living artists or protected IP.
- Verify hands, faces, product structure, packaging claims, text accuracy, logo placement, and regulatory claims.
- Treat AI output as exploration or production reference until reviewed by a human designer.
