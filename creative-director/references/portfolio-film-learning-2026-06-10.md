## 总体判断

这些成片补齐了脚本和项目资料之外的第三层信息：

`写法 -> 执行 -> 成片结果`

你的成片工作方式不是单一风格，而是根据项目类型在几种模式之间切换：

1. 明亮生活方式快消片。
2. 家庭/母婴/情绪实拍片。
3. 游戏/IP/明星大片感宣传片。
4. 产品功能证据片。
5. 动画/包装/二维信息片。
6. 汽车/数码/科技品牌片。

核心共性：

`画面要好看，但每个好看镜头都要承担人物、产品、情绪或卖点任务。`

## 成片层的核心审美

### 1. 真实质感优先

即使有 CG、包装或 AIGC，成片仍然常以真实人物、真实场景、真实光线作为落脚点。

常见画面：
- 家中窗光。
- 户外阳光。
- 露营/草地/山野/沙漠。
- 客厅、厨房、办公室、电竞房。
- 产品手持、使用、开关、屏幕反馈。

学习原则：

`特效可以进入画面，但产品和人的关系必须真实。`

### 2. 情绪比参数更早进入

快消、酒水、游戏、品牌片通常不是先说卖点，而是先建立情绪：

- 风花雪月：春天、花瓣、少女、山野、清透。
- 西夏啤酒：星空、沙漠、露营、朋友、音乐。
- Momcozy：产后疲惫、家庭空间、柔和暖光。
- OPPO：家庭关系、校园/生活回忆、人物情绪。
- 游戏片：队友、回流、热血、角色仪式。

学习原则：

`功能片先证据，情绪片先场域。`

### 3. 产品是故事道具，不是摆拍物

成片里产品经常通过动作进入：

- 拿起。
- 倒入。
- 举杯。
- 扫描。
- 插入。
- 点击。
- 拍照。
- 戴上。
- 推动。
- 开启。
- 跌落/耐摔测试。
- 清洁/拆装。

学习原则：

`产品不要只摆在桌上，必须被人物触发、使用、证明或传递。`

## 成片片型补充

### A. 明亮户外快消片

代表：
- 风花雪月。
- 西夏啤酒。
- 泰山原浆。
- 悦鲜活 X 永璞咖啡。

画面规律：
- 高亮自然光。
- 浅色服装。
- 花瓣、草地、山野、蓝天、云。
- 手持产品、举杯、开罐、倒酒。
- 产品和人物同时被阳光照亮。
- 结尾常有产品+情绪文案+品牌落版。

脚本启发：

`情绪场景 -> 产品动作 -> 群体/个人释放 -> 产品定版`

### B. 家庭情绪/母婴片

代表：
- Momcozy。
- 补肺丸春节/家庭版本。
- 暖风机。
- OPPO 重返校园。

画面规律：
- 室内自然窗光。
- 暖色或低饱和调色。
- 家庭关系和人物状态先行。
- 产品功能常作为生活改善出现，而非强硬售卖。
- 通过小孩、老人、妈妈、伴侣等关系制造情绪入口。

脚本启发：

`生活状态 -> 情绪压力 -> 产品介入 -> 状态改善 -> 柔和收尾`

### C. 游戏/IP/明星大片感

代表：
- PUBG MOBILE。
- MLBB。
- 梦幻西游 X 张艺兴。
- 英雄联盟手游 X 王俊凯。
- 火影忍者张杰 MV。
- 鸣潮周年庆。

画面规律：
- 强色彩和强光源。
- 舞台、城市夜景、虚拟空间、能量线。
- 明星/主角有清晰英雄 pose。
- CG 世界与实拍人物切换。
- 转场常靠能量、光、手机、屏幕、烟雾、角色动作完成。

脚本启发：

`现实锚点 -> 进入世界 -> 角色/明星仪式镜头 -> 爽点动作 -> 情绪高潮 -> 活动/品牌落版`

### D. 产品功能证据片

代表：
- TECNO CAMON 40 系列。
- 友基 UE16。
- TORRAS 充电头/香薰/手机壳。
- 亿联场景篇。
- 飘柔教学视频。

画面规律：
- 产品居中。
- 参数字卡明确。
- 屏幕 UI、使用动作、功能结果一一对应。
- 用对比、分屏、微距、特写降低理解成本。
- 常使用白底、黑底、科技蓝、干净桌面或真实使用场景。

脚本启发：

`产品亮相 -> 功能操作 -> 结果证明 -> 参数解释 -> 场景延展 -> 落版`

### E. 动画/包装解释片

代表：
- 早期寓言动画。
- 奶牛/咖啡二维包装。
- IP 角色包装。

画面规律：
- 冲突非常简单。
- 字幕/大字卡服务笑点或卖点。
- 角色表情和动作夸张。
- 画面信息以“看一眼能懂”为标准。

脚本启发：

`简单冲突 -> 角色动作 -> 反转/笑点 -> 信息落版`

## 镜头与执行习惯

### 常用镜头结果

- 产品手持特写。
- 人物微笑/回头/抬头。
- 低机位仰拍制造爽感。
- 俯拍展示场景关系。
- 主观视角进入手机/游戏/产品界面。
- 大场景建立情绪，小特写证明产品。
- 黑底或干净背景产品定版。

### 常用转场结果

- 酒液/水流进入下一镜。
- 手机屏幕进入虚拟世界。
- 能量线/光束穿梭。
- 烟雾/云/花瓣擦过镜头。
- 黑胶唱片/物件旋转转场。
- 产品掠过镜头做遮挡。

### 常用落版结果

- 产品 Hero + 主卖点。
- 活动信息 + LOGO。
- 参数字卡 + 法务小字。
- KV 风格定格。
- 黑底白字或干净浅底。

## 对脚本输出的新增要求

以后写脚本和分镜时，不只写“应该拍什么”，还要靠近这些成片结果：

1. 每条片子先判断成片应属于哪种视觉模式。
2. 为核心卖点设计真实动作，而不是只写字幕。
3. 情绪片必须先建立场域，再让产品进入。
4. 功能片必须给出操作、结果、参数、场景四层证据。
5. 游戏/IP片必须保留角色/明星的仪式感镜头。
6. 家庭片不要过度炫技，要保留表演和光线。
7. 产品定版必须明确：背景、产品角度、主卖点、LOGO或活动信息。
8. 如果涉及 CG/AI，必须在备注里标清它与实拍的衔接方式。

## 应用于 `6000LB 重力审判`

这支货架片应归入：

`产品功能证据片 + 工业测试事件片`

成片参考方向：

- 重量冲击要有物理感，不要纯 CG。
- 字幕和数字要像 TECNO / 友基类产品片一样干净、明确。
- 结构微距要像功能证据片一样清楚。
- 最终 Hero 可以用黑底或冷灰工业背景，产品居中，卖点一句话。

推荐成片结构：

`暗场产品 Hero -> 重物落下 -> MOCO遮挡升级 -> 结构微距证据 -> 功能操作补充 -> 6000LBS高潮 -> 黑底产品定版`

核心提醒：

`这支片不能拍成只有炫酷，要拍成看完就相信它真的能扛。`
