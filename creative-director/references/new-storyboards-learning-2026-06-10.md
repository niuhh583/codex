## 1. 西夏啤酒：情绪音乐型产品广告

基础信息：
- `# 西夏啤酒 #导演分镜`
- `DIR_SHOOTINGBOARD_V1`
- `Date:2024/11/28`
- `Film Time: 45min`，但实际镜头备注按 45s 左右节奏写。
- 19 个镜头，23 张分镜图。

核心结构：

`星空沙漠情绪开场 -> 产品从冰桶中被拿起 -> 朋友奔向露营地 -> 捧杯进入酒液/星空转场 -> 麦芽升级与醇爽升级 -> 音乐情绪高潮 -> 星空营地定版`

新增学习点：
- 情绪片里，产品卖点不直接硬讲，而是嵌入音乐、场景和情绪高潮。
- `夜空中最亮的星` 作为音乐记忆点，承担氛围和品牌情绪。
- 产品升级点用视觉化表达：`多加20%麦芽` 不是单独字幕，而是麦芽飞入环绕、酒液喷涌、星空叠画。
- 使用 `实拍+合成` 和 `实拍+CG` 标注生产边界。
- MOCO 用于产品英雄镜头和微距环绕，例如冰桶拿产品、巨大产品环绕、倒酒浪花。

可复用公式：

`情绪问题或情绪召唤 -> 产品成为参与仪式 -> 音乐/场景推高情绪 -> 卖点进入超现实视觉 -> 群体情绪释放 -> KV落版`

适合：
- 酒水、饮料、节庆、年轻社交、音乐主题、户外露营、品牌情绪片。

## 2. PUBG MOBILE：现实与游戏互切型回流宣传片

基础信息：
- `# PUBG MOBILE #导演分镜`
- `DIR_SHOOTINGBOARD_V1`
- `Date:2024/12/27`
- `Film Time: 100s`
- 33 个镜头，26 张分镜图。

核心结构：

`现实房间长镜头建立怀旧物件 -> 队友上线触发回流 -> 手机主观进入游戏 -> 游戏中连续出糗/冒险/重逢 -> 现实队友反应互切 -> 四人集合坐车冲桥 -> 回到现实 -> 活动奖励落版`

新增学习点：
- 开场长镜头不是炫技，而是建立回流情绪：地图、相册、沙漏、日历、唱片、队友上线。
- 现实和游戏的切换有明确动作媒介：手机主观、烟雾转场、降落伞划过、黑胶唱片转场。
- 游戏 CG 段不是单纯战斗，而是有喜剧失误和队友情绪：跳错地方、迷雾害怕、队友幸灾乐祸。
- 回流活动的商业信息藏在道具和结尾：日历、上线消息、OG套装、活动信息。
- 技术列被命名为 `技术+台词`，说明复杂片需要把实拍/CG/包装和台词放在同一执行字段。

可复用公式：

`现实怀旧锚点 -> 好友召回 -> 进入虚拟世界 -> 经典地图/经典错误/经典配合 -> 情绪重聚 -> 活动奖励收口`

适合：
- 游戏回流、IP周年、老玩家召回、多人联机、虚实融合宣传片。

## 3. Crazymar吹风机：美妆产品卖点证据链

基础信息：
- `Crazymar后进风高速吹风机｜导演分镜`
- `DIR_SHOOTINGBOARD_V1`
- `Date:2025/10/21`
- `Film Time: 40s`
- 26 个镜头，31 张分镜图。

核心结构：

`产品亮相 -> LED/冷热交互 -> 速干展示 -> 电机三维结构 -> 负离子微观可视化 -> 低噪 -> 屏幕动画 -> 配件分屏对比 -> 自动清洁提醒 -> 后进风清洁 -> 产品定版`

新增学习点：
- 美妆电器的卖点展示更强调“人体结果 + 微观原理”双证据。
- `2-5分钟速干` 通过 360 环绕和头发快速干爽表现。
- `18万转高速无刷电机` 进入三维内部结构展示。
- `5亿负离子` 用 AIGC 表现负离子缠绕头发、毛鳞片闭合。
- 配件效果用分屏对比：不带配件 / 扁头 / 卷发配件。
- `40小时自动清洁提醒` 通过屏幕亮起和后盖吹灰建立使用闭环。

可复用公式：

`外观高级亮相 -> 关键功能操作 -> 用户结果 -> 内部结构解释 -> 微观效果解释 -> 对比证明 -> 维护便利 -> 产品定版`

适合：
- 吹风机、美容仪、按摩仪、小家电、带屏幕/档位/配件的产品片。

## 对原有方法的补充

新增三种片型细分：

1. 情绪音乐型产品广告  
   不是先讲功能，而是先建立情绪场域，让产品成为仪式和情绪释放的道具。

2. 虚实互切游戏宣传片  
   用现实道具承接情绪，用游戏 CG 承接爽感，用活动奖励完成转化。

3. 美妆电器微观证据片  
   用人体结果证明效果，用三维/AIGC解释原理，用分屏对比降低认知成本。

## 后续执行规则补充

- 如果产品有强情绪场景，先判断能不能用音乐、群体、场景仪式承载卖点。
- 如果是游戏/IP项目，现实段必须有情绪锚点，虚拟段必须有爽点和角色关系。
- 如果是美妆/个护/小家电，必须同时设计外观、操作、结果、原理、对比和维护闭环。
- 对复杂 CG/AI 项目，备注列要写清 `实拍+包装`、`实拍+CG`、`素材+合成`、`三维`、`AIGC`。
- 转场媒介必须有物理或叙事理由：酒液、星空、手机、烟雾、黑胶、头发、屏幕、产品结构。
